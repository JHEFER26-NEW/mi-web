## Qué editar en index.html

- usuario real de Instagram

- correo real

- habilidades reales si deseas

## Subir a GitHub
1. Sube todos los archivos al repositorio
2. Ve a Settings > Pages
3. Activa GitHub Pages en la rama principal
4. Guarda y abre tu enlace

## Pruebas manuales
1. Abrir en computadora
2. Abrir en tablet
3. Abrir en celular
4. Revisar que la foto cargue
5. Revisar que los íconos de redes se vean
6. Revisar que los íconos de habilidades se vean
7. Probar enlaces de WhatsApp, Instagram, X y correo
